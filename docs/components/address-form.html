AddressForm

AddressForm is a form that capture an address. Essentially wraps AddressFields in a Form with submit and cancel buttons.

Not sure this is the right component for your use case? See other address-related components.

PropTypeDescriptionDefault
allowCancelboolShow the cancel button?true
cancel() => ?Function executed when the cancel button is clicked.
