history

history is the singleton instance of the history object passed to Router. Its main purpose is to allow calls to history.push(), history.replace(), and other methods without needing to use React Router’s withRouter HoC (source).

Example

import history from 'components/Routing/history'

const MyLink = ({ href, content }) =>
  <div onClick={() => history.push(href)}>
    {content}
  </div>
